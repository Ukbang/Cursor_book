What I do

3가지 주제로 빠르게, 재밌게, 결과까지.

주제 1

AI 활용 영상 컨텐츠 제작

나노바나나와 Grok을 이용해 홍보 컨텐츠를 빠르게 제작합니다.

  • 기획(후킹) → 콘티 → 초안 생성까지 한 번에
  • 브랜드 톤/타겟에 맞춘 표현 조정
  • 재사용 가능한 프롬프트/템플릿 정리

주제 2

AI 업무 자동화

n8n과 Opal로 일상 업무를 자동화해 ‘반복’을 줄입니다.

  • 업무 흐름을 쪼개서 자동화 포인트 찾기
  • 입력(폼/메일) → 처리 → 알림까지 연결
  • 실패/예외 케이스를 고려한 운영 방식

주제 3

AI와 함께하는 데이터 분석

Excel/CSV로 데이터 분석 대시보드를 만들고, 인사이트를 뽑아냅니다.

  • 지표 정의 → 정리 → 시각화 순서로 빠르게 구축
  • 보고서/대시보드 템플릿 형태로 재사용
  • “무엇을 봐야 하는지” 질문 설계 지원
